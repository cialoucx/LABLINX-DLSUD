Quick Actions

How It Works

1. Browse Equipment

Go to "Lab Equipment" to see all available items across Science, Sports, Facility, and Robotics labs.

2. Request or Reserve

Click "Request" on any available item. Fill in details like quantity and reason, then submit for admin approval.

3. Track & Return

Monitor deadlines on your dashboard. Return items on time to maintain a clean borrowing record.

4. Report Issues

Found damaged or missing equipment? File an incident report (LF-05) through the Report Dashboard tab.
